# AWS1-08: 設計パターン演習1 — 3層アーキテクチャ + Bastion Host

> **Mục tiêu:** Xây dựng kiến trúc 3-tier bảo mật cao: ALB (public) → Web Server (private) → Aurora (private), quản trị qua Bastion Host.

---

## Kiến trúc tổng quan

```
Internet
   │
[IGW: plab_igw]
   │
VPC: plab_vpc (10.11.0.0/16)
   │
   ├── PUBLIC SUBNETS
   │     ├── plab_pub_a (10.11.1.0/24 / us-east-1a)
   │     │     ├── Bastion Host (plab_bastion)  ← SSH từ Admin
   │     │     └── NAT Gateway A (EIP)
   │     │
   │     └── plab_pub_b (10.11.3.0/24 / us-east-1b)
   │           └── NAT Gateway B (EIP)
   │
   ├── PRIVATE WEB SUBNETS
   │     ├── plab_pri_a (10.11.2.0/24 / us-east-1a)
   │     │     └── Web Server 1 (plab_websv_a)
   │     └── plab_pri_b (10.11.4.0/24 / us-east-1b)
   │           └── Web Server 2 (plab_websv_b)
   │
   └── PRIVATE DB SUBNET
         └── plab_pri_rds (10.11.5.0/24 / us-east-1c)
               └── Aurora MySQL (plab-db)

ALB (plab-elb) ← spanning plab_pub_a + plab_pub_b → Target Group → Web Servers
```

### SSH vào Web Server 1 qua Bastion (Jump Host)

```bash
# 1. Đẩy .pem key lên Bastion (Tera Term SCP)
# 2. Trên Bastion:
chmod 600 [key].pem
ssh ec2-user@[Private_IP_websv_a] -i [key].pem

# 3. Cài Apache + PHP
sudo yum install -y httpd php php-mysqli
sudo systemctl start httpd
sudo systemctl enable httpd

# 4. Tạo file web hello.php
sudo vi /var/www/html/hello.php
```
